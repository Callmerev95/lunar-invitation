# ROADMAP - Lunar Invitation

## Project Phases

### Phase 1: Multi-Template Website (Saat Ini)

**Target Waktu:** 3-4 minggu

**Deliverables:**

- 5 Template undangan berbeda
- Single Page Application yang smooth & elegant
- RSVP Form + Google Maps
- Music Player + Countdown
- QR Code + Share Button
- Mobile-first & High Performance
- Dokumentasi lengkap

**Success Criteria:**

- Loading time < 1.5 detik
- Lighthouse Score > 95
- Smooth animation di semua device
- 5 template siap dipakai

---

### Phase 2: Semi SaaS (2-4 minggu setelah Phase 1)

**Fitur Tambahan:**

- Admin dashboard sederhana
- Client bisa customize data sendiri (nama, foto, tanggal, dll)
- Template switcher
- Preview real-time
- Auto generate QR Code
- Download undangan sebagai PDF

**Teknologi Tambahan:**

- Database (Supabase / Vercel Postgres)
- Authentication (NextAuth / Clerk)
- Form builder dinamis

---

### Phase 3: Full SaaS (Opsional - 1-2 bulan)

**Fitur:**

- Self-service full (client login → pilih template → edit → publish)
- Payment Gateway (Midtrans / Stripe)
- Subscription model atau per undangan
- Analytics (berapa tamu konfirmasi, dll)
- Custom domain untuk client premium
- Guest Book + Digital Envelope

**Monetisasi:**

- Paket Basic, Premium, Luxury
- Template marketplace

---

### Phase 4: Future Expansion

- Multi language (English + Indonesia)
- Live streaming integration
- Family tree / seating chart
- AI wedding assistant
- Mobile App version

---
