# PROJECT BRIEF - Lunar Invitation

## Project Overview

Website undangan pernikahan digital premium dengan konsep **Minimalist Luxury**. Dibuat untuk memenuhi kebutuhan pasangan kekinian yang menginginkan undangan yang elegan, modern, interaktif, dan mudah dibagikan.

**Project Name**: Lumina Invitation  
**Tagline**: "Elegant Digital Wedding Invitation"

## Objectives

- Membuat undangan digital yang premium dan memorable
- Memberikan pengalaman terbaik bagi tamu undangan
- Membangun portofolio dan produk yang bisa dijual (Multi-Template)
- Menjadi fondasi untuk pengembangan SaaS di masa depan

## Target Audience

- Pasangan usia 25-35 tahun
- Suka estetika minimalis, elegan, dan mewah
- Ingin undangan yang personal, interaktif, dan kekinian
- Kebanyakan mengakses via smartphone

## Core Features (Versi 1)

### Must Have

- Hero Section dengan foto couple + nama pasangan + tanggal
- Countdown Timer ke hari H
- Detail Acara (Akad Nikah & Resepsi)
- Google Maps + tombol "Buka di Maps"
- Galeri Foto (Couple & Prewedding)
- Timeline Acara
- RSVP Form + Konfirmasi Kehadiran
- Music Player (lagu romantis)
- QR Code Undangan
- Share Button (WhatsApp, Instagram, dll)

### Nice to Have (akan ditambahkan bertahap)

- Guest Book
- Digital Envelope / Amplop Digital
- Dresscode & Protokol
- Multi Language (Indonesia & English)

## Technical Requirements

- Framework: Next.js 15 (App Router)
- Styling: Tailwind CSS + shadcn/ui
- Animation: Framer Motion
- Form: React Hook Form + Zod
- Deployment: Vercel
- Mobile First & Super Smooth Scrolling
- Fast Loading (Performance Score > 95)

## Design Direction

- Vibe: Minimalist Luxury, Soft Romantic, Elegant
- Warna utama: Soft Gold, Warm Cream, Deep Burgundy, Sage Green
- Typography: Elegant serif untuk heading + Clean sans-serif untuk body

## Deliverables (Versi 1)

- 4-5 Template undangan berbeda
- Admin page sederhana untuk customize data
- Dokumentasi cara penggunaan
- Source code yang rapi dan scalable

## Phase Pengembangan

**Phase 1** → Multi-Template Website (saat ini)
**Phase 2** → Semi SaaS (client bisa edit sendiri)
**Phase 3** → Full SaaS + Payment Gateway
