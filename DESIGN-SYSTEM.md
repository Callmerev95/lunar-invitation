# DESIGN SYSTEM - Lunar Invitation

## Project Overview

Website undangan pernikahan digital premium dengan konsep **Minimalist Luxury**. Elegant, soft, romantic, dan modern.

## 1. Brand Personality

- Elegant & Timeless
- Warm & Romantic
- Sophisticated tapi tidak norak
- Minimalis dengan sentuhan mewah

## 2. Color Palette

### Primary Colors

- **Primary Gold**: `#D4AF88` (Soft Gold)
- **Primary Cream**: `#F5F0E8` (Warm White)
- **Accent Burgundy**: `#8B4F6F` (Deep Romantic)
- **Accent Sage**: `#A8B5A2` (Soft Natural)

### Neutral Colors

- **Background**: `#FAF7F2`
- **Text Dark**: `#3A2F2F`
- **Text Light**: `#6B5E5E`
- **White**: `#FFFFFF`

### Gradient Examples

- Gold to Cream: `linear-gradient(135deg, #D4AF88 0%, #F5F0E8 100%)`
- Burgundy to Gold: `linear-gradient(135deg, #8B4F6F 0%, #D4AF88 100%)`

## 3. Typography

### Heading Font (Elegant)

- **Font Family**: `Playfair Display` / `Cinzel`
- **Weights**: 400, 500, 600, 700
- **Style**: Serif

### Body Font (Readable)

- **Font Family**: `Inter` / `Satoshi`
- **Weights**: 400, 500, 600
- **Style**: Sans-serif

### Font Sizes

- Hero Title: 4.5rem (desktop) / 3rem (mobile)
- Section Title: 2.5rem / 2rem
- Body Text: 1.125rem
- Small Text: 0.95rem

## 4. Spacing & Layout

- Container max-width: 1280px
- Section padding: 6rem (desktop) / 4rem (mobile)
- Border radius: 16px - 24px (soft)
- Box shadow: soft & elegant (`0 10px 30px rgba(0,0,0,0.08)`)

## 5. Animation Style

- Smooth & delicate
- Duration: 0.8s - 1.2s
- Easing: `cubic-bezier(0.4, 0, 0.2, 1)`
- Trigger: Scroll + Hover

## 6. Component Design Rules

- Button: Soft rounded, gold accent, elegant hover
- Card: Soft shadow, warm background, generous padding
- Navigation: Minimal & transparent
- Music player: Elegant minimal control
